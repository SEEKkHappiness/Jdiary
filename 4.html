HTML 태그 설명

  1. HTML= Hyper Text Markup Language(초본문표식달기언어).
    ->웹페이지를 위한 지배적인 마크업 언어.
  2. h1 ~ h6 = heading ->제목 크기
  3. u = underline -> 글자에 밑줄
  4. strong -> 글자 굵게
  5. img = image -> 이미지
  6. src = source -> 불러올 이미지 파일
  7. width -> 사진 크기 정하기
  8. p = paragraph -> 단락 나누기
  9. br = breaker -> 단락 나누기
  10. li = list -> 목차(목록)
  11. ol = ordered list (숫자를 나열해줌) -> 목차의 아버지 태그
    ul = unordered list (기호로 목차를 나열해줌) -> 목차의 아버지 태그
  12. title -> 웹 홈페이지 제목 달기
  13. charset = character(문자) set(배열)
  14. body -> 본문
  15. head -> 본문 설명
  16. doctype -> 이 문서는 html에 있다는 뜻 (html 태그 위에 관용적으로 달아야함)
  17. html -> 모든 태그의 고위층 태그
